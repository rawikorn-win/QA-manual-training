## **Test case#1: Facebook Log In**

**Test Case ID:** FACEBOOK_Login_001  
**Title:** Verify Facebook Login with valid email and password
**Precondition:** User has a valid email: testest123123@gmail.com , password: 123123
**Steps to Reproduce:**
1. Enter Facebook site 
2. Enter valid email and password
3. Click Log in button

**Expected Result:**  
User successfully logged in

**Actual Result:**  
User successfully logged in

---
**Test Case ID:** FACEBOOK_Login_002
**Title:** Verify Facebook Login with invalid email
**Precondition:** User has a invalid email: testest123123@gmail.com ,and valid password: 123123
**Steps to Reproduce:**
1. Enter Facebook site 
2. Enter invalid email and valid password
3. Click Log in button

**Expected Result:**  
The email address or mobile number you entered isn't connected to an account.

**Actual Result:**  
The email address or mobile number you entered isn't connected to an account.
